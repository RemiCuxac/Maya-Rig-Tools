# Maya-Rig-Tools

##  Description
A toolset of rigging tools for maya users.

The purpose of this project is to provide a flexible environment for your rigging projects, that can be installed and uninstalled if needed.
You can download the full project and install it to your computer, or simply get what you want.

- Scripts are designed to be standalone, so they doesn't need any dependencies. 
- Tools are designed mostly with an interface an may need external modules like numpy, scipy, so they may require to install some of them to be run. That's why they're provided with a DragAndDropInstall.py that wil do all the job for you. 
- Shelves files contains some of the most needed scripts, and maya usefull commands. 

Feel free to send a message or create pull requests if you find any bugs 🪰!
